For the back end, please download the repository from the following url https://github.com/Mgal0127/my-app-backend.git

Open the folder with visual studio code, navigate to my-app-backend or my-app-backend-master and in the terminal run the following commands:

npm install -g @babel/core @babel/cli
npm install
npx babel-node src/server.js

For the front end, please download the repository from the following url https://github.com/Mgal0127/my-app.git
Open the folder with visual studio code, navigate to my-app or my-app-master and in the terminal run the following commands:

npm install
npm start

The project should now be deployed onto the explorer
